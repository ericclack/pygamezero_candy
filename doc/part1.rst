Part 1
======

Getting Started
---------------

- Press the **New** button in Mu to open a new file and enter the following lines:

.. code:: python

    WIDTH = 400
    HEIGHT = 560
    TITLE = 'Candy Crush'

- Press **Save** and save the file as candy.py

- press **Play** to see what this code does.

You should see an new empty window appear.
